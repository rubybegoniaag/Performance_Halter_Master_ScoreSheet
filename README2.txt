# Performance Halter Score Calculator

This project is an online calculator designed to score horses in a performance halter class. It allows users to select scores for various categories and calculates the total score dynamically.

## Features

- Select scores for each category using radio buttons.
- Apply penalties using checkboxes.
- Calculate the total score automatically.
- User-friendly interface.
